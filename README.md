# Dashboard de Inventario Arbóreo - Campus Omar Dengo
## Universidad Nacional de Costa Rica (UNA)

Dashboard web interactivo para visualizar y analizar el inventario arbóreo del Campus Omar Dengo, con estimaciones de biomasa y captura de carbono forestal.

**App publicada:** https://carbono-campus-omar-dengo-una-ugybcv7sw8upkyyck5ltte.streamlit.app

---

## Archivos del proyecto

| Archivo | Descripción |
|---|---|
| `app.py` | Código principal del dashboard |
| `requirements.txt` | Dependencias de Python |
| `CENSO_Historico_UNA_con_calculos.xlsx` | Base de datos del inventario (2018–2025) |
| `.streamlit/config.toml` | Tema y configuración visual |
| `DOCUMENTACION.md` | Documentación completa del proyecto |

---

## Ejecución local

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Tecnologías

Python · Streamlit · Pandas · Plotly · OpenPyXL
